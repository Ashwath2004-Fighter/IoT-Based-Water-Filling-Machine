About the Project

Addressing water wastage through intelligent automation and IoT technology

The Problem

Water scarcity is one of the most pressing global challenges. In households, industries, and public spaces, improper water dispensing leads to significant wastage. Manual filling operations result in overflow, imprecise quantities, and constant monitoring needs.

Our Solution

The IoT Based Water Filling Machine automates the entire water dispensing process using smart sensors and microcontrollers. It ensures precise water quantity, prevents overflow, and provides real-time monitoring capabilities through IoT connectivity.

Working Principle

Step-by-step explanation of how the system operates

01

User Input

The user enters the desired quantity of water through the keypad interface. The system displays the input on the LCD screen for confirmation. Users can also select preset quantities for convenience.

02

Container Detection

The ultrasonic sensor detects the presence and position of the container. It calculates the available space and ensures the container is properly placed before initiating the filling process.

03

Pump Activation

Once confirmed, the microcontroller sends a signal to the relay module, which activates the water pump. Water begins flowing from the reservoir to the container at a controlled rate.

04

Real-time Monitoring

The ultrasonic sensor continuously monitors the water level in the container. The LCD displays real-time information including current water level, filling progress, and system status.

05

Safety Check

The system continuously monitors the container level via the ultrasonic sensor to prevent overflow. This safety mechanism ensures maximum protection and precision throughout the filling process.

06

Automatic Shutdown

When the desired quantity is reached or the container is nearly full (detected by ultrasonic sensor), the microcontroller deactivates the pump through the relay module. The system beeps to indicate completion and displays the total filled status.

07

IoT Data Transmission

The ESP8266 module transmits usage data to the cloud platform. Users can monitor water consumption patterns, receive notifications, and control the system remotely through a mobile application or web dashboard.
